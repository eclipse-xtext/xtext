## Eclipse Integration

## xtext-maven-plugin

### testGenerate Goal

The maven plugin has now two goals, generate and testGenerate. The new testGenerate goal runs during the generate-test-sources phase and uses the test class path of the maven project.

## Upgrades

* Xtext 2.21 now requires EMF 2.20 as minimal required version.
